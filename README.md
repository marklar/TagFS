
# Using TagFS


## Deploying TagFS

First, install TagFS: [Install](docs/Install.md).

Then run it:

1. Modify env vars in shell script `./run` as desired.
2. `$ chmod +x run`
3. Run it: `$ ./run`

And finally, go to the TagFS volume: `$ cd <your-mount-point>` (=
`$TAGFS_MOUNT_POINT`).
